## App Review Information

### Review Notes (paste this into the "Notes" field for Apple reviewers)

```
SwipeCat is a product-discovery app that displays Amazon product listings sourced via the Rainforest API (a licensed Amazon data provider). Users swipe through cards to like, pass, or save products. Tapping a product opens the Amazon.com product page in Safari/the Amazon app.

Monetization: SwipeCat participates in the Amazon Associates affiliate program. Product links include our Associates tracking tag (swipecat-20). This is disclosed on the home screen, the About page (/about), and the Privacy Policy (/privacy).

No account is required to use the app. User preferences (liked/saved/passed items) are stored locally on-device using localStorage and are never transmitted to our servers.

The /admin route is a catalog management tool for the app owner only. It requires Supabase authentication and an admin role — it is not accessible to regular users and is not linked from any user-facing navigation.

Test credentials are not required as all app functionality is available without signing in.
```
